# Code Review System App

## Описание проекта

Code Review System App — клиентская часть информационной системы для организации процесса ревью кода в разработке программного обеспечения.

Приложение разработано с использованием технологии WPF и подключено к базе данных Microsoft SQL Server.

## Выполненная работа

В рамках практической работы были реализованы:

- окно авторизации пользователей;
- окно регистрации пользователей;
- подключение к MSSQL;
- авторизация через базу данных;
- регистрация новых пользователей;
- проверка существования пользователя;
- навигация между страницами;
- использование Entity Framework (ADO.NET Entity Data Model);
- централизованное хранение стилей и ресурсов в App.xaml;
- фирменный стиль приложения;
- подключение логотипа и иконки проекта.

## Изменения в практической работе №15

В структуру базы данных было добавлено хранение изображений в бинарном формате BLOB.

В таблицу PullRequests добавлено поле:

- ImageData (VARBINARY(MAX))

Поле предназначено для хранения изображений, связанных с pull request и изменениями кода.

Также были обновлены тестовые данные и SQL-скрипт инициализации базы данных.

## Практическая работа №16

В рамках практической работы была реализована страница каталога сущностей системы в формате карточек.

Для отображения данных используется элемент ListView с WrapPanel, формирующий сетку карточек.

Карточки содержат:

- изображение;
- название pull request;
- описание;
- статус;
- дополнительные характеристики.

Данные подгружаются из базы данных MSSQL через Entity Framework.
