# Code Review System

## Описание проекта

Система ревью кода предназначена для организации процесса проверки исходного кода в процессе разработки программного обеспечения.

Система позволяет:

- создавать pull requests;
- назначать ревьюверов;
- хранить комментарии к изменениям;
- отслеживать качество кода;
- анализировать метрики покрытия тестами и сложности кода;
- интегрировать статусы CI/CD.

## Предметная область

Основные сущности системы:

- Разработчики
- Проекты
- Pull Requests
- Комментарии
- Метрики качества
- CI/CD статусы

База данных реализована в Microsoft SQL Server.

## Используемые технологии

- MSSQL
- SQL
- GitHub

## Структура базы данных

База данных содержит следующие таблицы:

- Roles
- Developers
- Projects
- PullRequests
- ReviewAssignments
- Comments
- QualityMetrics
- CICDStatuses
