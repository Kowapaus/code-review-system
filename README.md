# Code Review System App

## Описание проекта

Code Review System App — клиентская часть информационной системы для организации процесса ревью кода в разработке программного обеспечения.

Приложение разработано с использованием WPF и Microsoft SQL Server. Система позволяет выполнять авторизацию пользователей, управлять pull request-записями и отображать данные в формате карточек.
